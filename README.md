# kruspix

## Introduction

Kruspix is a hands-on, educational kernel for the Raspberry Pi, written in Rust. This project is designed to help you get a feel for bare metal programming and build your own operating system from the ground up.

## Prerequisites

- [Rust](https://www.rust-lang.org/): Make sure you have Rust installed.
- [QEMU](https://www.qemu.org/): Required for emulating the Raspberry Pi and testing the kernel without real hardware.
